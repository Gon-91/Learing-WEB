How to connect to the internet?

앞의 페이지에서 글처럼 우리가 본 처음의 웹 그리고 지금 이 웹페이지도 인터넷 상에 올라와서 우리가 보고 쓸수 있게 된 것입니다. 자, 그러면 어떻게 우리가 만든 웹을 거대한 통신시스템인 인터넷에 올릴수 있는 걸까요?


Server and Client / Web Server and Web Browser

서버와 클라이언트, 웹 서버와 웹 브라우저 라고 불리며 이 구성원이 인터넷이 연결되어 서로의 상호 작용을 하여 우리의 웹페이지들을 볼수 있도록 해줍니다. 여기서, 서버 또는 웹 서버라고 불리는 이곳에서 우리가 어떠한 언어로 만든 웹페이지를 저장하고 주소를 정의하여 정보를 보낼 준비를 하는 곳입니다. 웹 서버는 정보를 보내주는 역할의 장치입니다. 그리고 클라이언트 웹 브라우저라고 부르는 이녀석은 서버에서 정의된 주소로 요청을 하는 역할을 하게됩니다. 최종적으로 정리하자면, 상호간의 컴퓨터에 인터넷이 연결되어있다면, 정보를 제공해주는 쪽은 서버 또는 웹서버라 불리며 정보를 요청하는 곳은 브라우저 또는 클라이언트라 불르며 이런 구성원을 통하여 우리가 인터넷을 통해 페이지를 만들고 볼수있게 된 것입니다.


Server를 일반인들이 사용 가능 할까 ?


Server hosing

사실 서버라는 개념을 이해했다고 해도 일반인들이 서버를 구축하고 운영하는것은 상당이 난이도가 높은 일입니다. 이런 서버를 구축하고 대신 구동해주는 업체들이 많이 있습니다. 이러한 업체들을 web server hosing 서버라고 불리웁니다. 이 중에 제가 이 글을 올리고 있는 이곳, Github 같은 곳이 대표적인 사이트 입니다 무료 이런 컨텐트 들을 이용할수 있게 말이죠.
